About Fax Test Tool

Fax Test Tool is designed to send a standard test page to a destination fax endpoint and report the provider's delivery status. It is useful for setup checks, vendor onboarding, and confirming that a fax line is ready to receive transmissions.

Delivery status comes from fax transmission events. A delivered status means the transmission was reported as delivered by the provider, not that a human reviewed the page.

Why a generic test page matters

Fax testing should avoid private documents. A neutral generated page gives administrators a consistent signal while reducing the chance that sensitive data is sent to the wrong destination during setup.
